# dotfiles

## Requirements

- Recent version of zsh
- zplug [here](https://github.com/zplug/zplug)

## Other stuff

- Recent ViM with Vundle
- tmux
- alacritty (fast gl based terminal)
- VS Code

```zsh
git clone https://github.com/radiosilence/dotfiles ~/.dotfiles
cd ~/.dotfiles
./install.zsh
```

Only symlinks .hidden files
